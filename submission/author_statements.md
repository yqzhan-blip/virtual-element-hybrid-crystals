# Author Statements

## Competing Interests

The authors declare no competing interests.

## Author Contributions

Y.Z. conceived the project, designed the virtual-element framework, implemented the model extensions and decoding pipeline, performed generation and DFT validation, analyzed the data, and wrote the manuscript.

## Data Availability

All source code, generated structures, analysis scripts, and DFT input templates are available at https://github.com/yqzhan-blip/virtual-element-hybrid-crystals. The training dataset was constructed from publicly available hybrid crystal structures and synthetically generated combinations. Pretrained MatterGen weights are available through Microsoft Research.

## Code Availability

Modified code including ExtendedAtomEmbedding, virtual element definitions, decoding scripts, and analysis pipelines is available at https://github.com/yqzhan-blip/virtual-element-hybrid-crystals under the MIT License.

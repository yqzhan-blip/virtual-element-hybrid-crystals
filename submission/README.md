# Submission Package for *npj Computational Materials*

This directory contains materials for submitting the manuscript **"A virtual-element framework for generative design of organic–inorganic hybrid crystals"** to *npj Computational Materials*.

## Files

- `cover_letter.md` — Cover letter to the Editor
- `author_statements.md` — Competing interests, author contributions, data/code availability, funding
- `submission_checklist.md` — Pre-submission checklist

## Main Manuscript Files

Located in `../paper/`:

- `paper_draft_v5.docx` — Main manuscript (Word)
- `SI_v5.docx` — Supplementary Information (Word)
- `paper_draft_v5.md` — Main manuscript (Markdown source)
- `SI_v5.md` — Supplementary Information (Markdown source)
- `figures/*.png` — Main manuscript figures (Figure 1–5)
- `figures_png/*.png` — Supplementary figures (Figure S1–S20)

## Submission System

Submit via the *npj Computational Materials* Editorial Manager site:
https://www.nature.com/npjcompumats/for-authors
